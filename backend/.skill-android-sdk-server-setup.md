# Android SDK 服务器环境部署

## 触发条件
用户要求在无图形界面的Linux服务器上安装Android开发环境，用于编译打包APK。

## 前置检查

```bash
free -h && nproc && df -h /
```

## ⚠️ 关键：低内存服务器必须先创建Swap

Android编译工具内存密集。RAM < 2GB 时，**先创建swap再装任何包**：
```bash
dd if=/dev/zero of=/swapfile bs=1M count=2048
chmod 600 /swapfile
mkswap /swapfile
swapon /swapfile
echo '/swapfile swap swap defaults 0 0' >> /etc/fstab
```

## 安装步骤

### 1. 安装JDK
```bash
apt-get update && apt-get install -y openjdk-17-jdk-headless
```

### 2. 下载SDK Command Line Tools
```bash
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export ANDROID_HOME=/opt/android-sdk
mkdir -p $ANDROID_HOME/cmdline-tools
cd /tmp
wget -q https://dl.google.com/android/repository/commandlinetools-linux-11076708_latest.zip -O cmdline-tools.zip
unzip -q cmdline-tools.zip -d /tmp/ctmp
mv /tmp/ctmp/cmdline-tools $ANDROID_HOME/cmdline-tools/latest
rm cmdline-tools.zip && rm -rf /tmp/ctmp
```

### 3. 接受许可证（关键步骤）
许可证需要交互式输入——反复管道 `y`：
```bash
export PATH=$ANDROID_HOME/cmdline-tools/latest/bin:$PATH
printf "y\ny\ny\ny\ny\ny\ny\ny\ny\n" | sdkmanager --sdk_root=$ANDROID_HOME --licenses
```
出现 "All SDK package licenses accepted" 才算成功。如果没成功，多传几个 `y`。

### 4. 安装SDK组件
```bash
sdkmanager --sdk_root=$ANDROID_HOME \
  "platforms;android-34" \
  "build-tools;34.0.0" \
  "platform-tools"
```

### 5. 配置环境变量
添加到 `~/.bashrc`：
```bash
export JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64
export ANDROID_HOME=/opt/android-sdk
export PATH=$ANDROID_HOME/cmdline-tools/latest/bin:$ANDROID_HOME/platform-tools:$JAVA_HOME/bin:$PATH
```

## 验证
```bash
java -version && javac -version && adb version && sdkmanager --version
sdkmanager --list --sdk_root=$ANDROID_HOME
```

## 故障排查

| 问题 | 解决 |
|------|------|
| "licenses not accepted" | 多传几个 `y`，有时有额外许可证 |
| 编译时OOM | 确保swap激活；Gradle JVM参数调小 |
| SDK包找不到 | `sdkmanager --list` 看可用版本 |
| 下载超时 | SDK下载慢，增加timeout或重试 |

## Gradle内存调优（1核机器必备）

在项目根目录创建 `gradle.properties`：
```properties
org.gradle.jvmargs=-Xmx512m -XX:MaxMetaspaceSize=256m
org.gradle.parallel=false
org.gradle.caching=true
```

## 实战经验（2026-06）
- **许可证接受**：`sdkmanager --licenses` 是交互式程序，不能简单用 `yes |`，必须 `printf "y\n"` 逐行发送
- **SDK工具版本**：commandlinetools-linux-11076708 是较新版本，如果下载失败可尝试 https://developer.android.com/studio#command-tools 找最新链接
- **磁盘占用**：完整SDK约450MB+，确保至少有1GB可用空间
- **编译速度**：1核机器首次编译可能需要10-30分钟，后续增量编译会快很多
